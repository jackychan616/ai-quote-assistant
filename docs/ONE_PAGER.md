# AI Quote + Follow-up Assistant (One Pager)

## 問題
香港中小企常見：
- 查詢多，但覆得慢
- 報價後客戶失蹤
- 跟進靠人手，容易漏

## 解決方案
一個 web app，打通：
1. Lead 管理
2. Quote 建立（多項目）
3. Follow-up 排程（手動 + 自動）
4. Dashboard KPI

## 核心價值
- 減少漏單
- 縮短回覆時間
- 提升報價後跟進完成率

## 目標客群
- 美容院
- 補習社
- 裝修公司
- Marketing agency

## 7日 Pilot Offer（建議）
- Setup: HK$1,500
- Pilot: HK$1,500（可扣月費）
- KPI：回覆率、預約率、跟進完成率

## Demo 路線
- `/lead/new` -> `/quote/:id` -> `/followups` -> `/dashboard`
