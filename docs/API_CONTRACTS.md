# API Route Contracts (MVP Draft)

Base URL: `/api`
Content-Type: `application/json`

## GET `/api/health`
### Response 200
```json
{ "ok": true, "service": "ai-quote-assistant-web", "timestamp": "2026-02-27T12:00:00.000Z" }
```

## Leads

## GET `/api/leads`
Query:
- `page` number (default 1)
- `pageSize` number (default 20, max 100)
- `status` optional enum: `new|qualified|quoted|won|lost`
- `q` optional keyword (name/email/company)

Response 200:
```json
{
  "items": [
    {
      "id": "uuid",
      "fullName": "Jane Doe",
      "email": "jane@acme.com",
      "status": "new",
      "company": "Acme",
      "createdAt": "ISO"
    }
  ],
  "pagination": { "page": 1, "pageSize": 20, "total": 1 }
}
```

## POST `/api/leads`
Body:
```json
{
  "fullName": "Jane Doe",
  "email": "jane@acme.com",
  "phone": "+886...",
  "company": "Acme",
  "source": "web_form",
  "notes": "Interested in annual plan"
}
```
Response 201: Lead object

## Quotes

## GET `/api/quotes`
Query:
- `leadId` optional uuid
- `status` optional enum: `draft|sent|accepted|rejected|expired`

Response 200: paginated quote list

## POST `/api/quotes`
Body:
```json
{
  "leadId": "uuid",
  "currency": "TWD",
  "lineItems": [
    { "name": "Setup", "qty": 1, "unitPrice": 12000 },
    { "name": "Monthly Subscription", "qty": 12, "unitPrice": 3000 }
  ],
  "discountAmount": 0,
  "taxRate": 0.05,
  "validUntil": "2026-03-15",
  "generateAiSummary": true
}
```
Response 201:
```json
{
  "id": "uuid",
  "leadId": "uuid",
  "subtotal": 48000,
  "taxAmount": 2400,
  "totalAmount": 50400,
  "status": "draft",
  "aiSummary": "..."
}
```

## Follow-ups

## GET `/api/followups`
Query:
- `status` optional `pending|done|cancelled`
- `dueBefore` optional ISO datetime
- `assigneeUserId` optional uuid

Response 200: paginated followup list

## POST `/api/followups`
Body:
```json
{
  "leadId": "uuid",
  "quoteId": "uuid",
  "dueAt": "2026-03-01T09:00:00Z",
  "channel": "email",
  "priority": "high",
  "messageDraft": "Checking if you have any questions about the quote.",
  "autoGenerated": true
}
```
Response 201: followup object
